Static List in Adobe Marketo Engage
Overview
Adobe describes Static Lists as one of the most simple and useful features in Marketo. A Static List is a list of names from the Marketo database whose membership is based on adding or removing each person rather than on defined rules.
Difference from Smart Lists
Adobe explicitly contrasts Static Lists with Smart Lists. Smart Lists are based on defined rules, while Static Lists are based on direct membership management.
How people are added or removed
Adobe lists several ways to add or remove people from a Static List, including Smart Campaign flow steps, single flow actions, dragging people into a list, and list import. This makes Static Lists useful when campaign operators need stable, intentional membership rather than automatically refreshed criteria.
Use cases
Adobe gives examples such as a pre-selected list for a marketing message, a competitors list, or a temporary operational list that Smart Campaigns later update. Static Lists are therefore valuable for controlled audience handling, especially when list membership should not change automatically as person data changes.
